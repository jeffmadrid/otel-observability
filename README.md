# otel-observability

## Getting started

### Kubernetes setup

Start up your own Kubernetes (`minikube` in this case)

```shell
minikube start
minikube dashboard
```

### Build

Convenience scripts to simplify build, deploy and undeploy.

```shell
./build.sh
```

### Deploy

```shell
./deploy.sh
```

### Undeploy

```shell
./undeploy.sh
```
